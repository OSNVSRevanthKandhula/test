# Personal Portfolio Website

A modern and responsive personal portfolio website created to showcase my skills, projects, resume, and contact information. This portfolio highlights my background in Mechanical Engineering, CAD design, data analysis, and web development.

## 🚀 Features

* Responsive and modern UI design
* About Me section
* Skills section
* Project showcase
* Resume download/open in new tab
* Contact section with direct email integration
* Smooth navigation and clean layout
* GitHub Pages deployment support

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Grid Layout
* GitHub Pages

## 📦 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/yourusername/repository-name.git
```

2. Open the project folder

```bash
cd repository-name
```

3. Run the `index.html` file in your browser.

## 📤 Deployment on GitHub Pages

1. Upload all project files to your GitHub repository.
2. Go to **Repository Settings**.
3. Open **Pages** section.
4. Select the branch as `main`.
5. Save and wait a few seconds.
6. Your portfolio website will be live.
